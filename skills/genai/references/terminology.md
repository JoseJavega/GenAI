# Glosario de Términos Genealógicos (Español)

## Tipos de Registro

| Término | Equivalente | Contexto |
|---------|-------------|----------|
| Registro Civil | Civil Registry | Registros vitales oficiales |
| Registro parroquial | Parish record | Registros de la iglesia |
| Partida de bautismo | Baptismal certificate | Iglesia |
| Partida de matrimonio | Marriage certificate | Iglesia |
| Partida de defunción | Death certificate | Iglesia |
| Libro de bautismos | Baptismal register | Iglesia |
| Libro de matrimonios | Marriage register | Iglesia |
| Libro de defunciones | Death/burial register | Iglesia |
| Confirmación | Confirmation | Sacramento |
| Velaciones | Wedding blessing | Iglesia (a veces por separado del matrimonio) |
| Expediente matrimonial | Marriage investigation file | Archivo diocesano |
| Acta de nacimiento | Birth certificate | Civil |
| Acta de matrimonio | Marriage certificate | Civil |
| Acta de defunción | Death certificate | Civil |
| Esquela | Obituary notice | Prensa |
| Fe de bautismo | Baptismal certificate | Iglesia |
| Fe de vida | Proof of life | Cualquier registro |
| Fe de soltería | Certificate of single status | Iglesia/civil |

| Diócesis | Diocese | Distrito eclesiástico |
| Archivo Histórico Provincial (AHP) | Provincial Historical Archive | Registros notariales, administrativos |
| Archivo Municipal | Municipal/Town Archive | Registros locales |
| Archivo Diocesano | Diocesan Archive | Registros parroquiales consolidados |
| Archivo de Protocolos | Notarial Archive | Registros notariales |
| Juzgado de la Paz | Justice of the Peace Court | Registros civiles |
| Oficina del Registro Civil | Civil Registry Office | Registros vitales |
| Ayuntamiento | Town Hall/City Council | Gobierno local |
| Padrón | Census/population register | Municipal |
| Catastro de Ensenada | Ensenada Cadastre | Censo/topografía siglo XVIII |
