# Recursos de Genealogía Española

## Archivos Principales

### PARES - Portal de Archivos Españoles
- **URL**: https://pares.culturaydeporte.gob.es
- **Contenido**: Archivos estatales digitalizados (Simancas, Indias, Histórico Nacional)
- **Búsqueda avanzada**: Filtros por texto, archivo, rango de fechas, tipo de documento

### Biblioteca Nacional de España (BNE)
- **Hemeroteca Digital**: https://hemerotecadigital.bne.es (110M+ páginas digitalizadas)
- **Biblioteca Digital Hispánica**: https://bnedigital.bne.es
- **Catálogo General**: https://catalogo.bne.es

### FamilySearch
- **URL**: https://www.familysearch.org
- **Registros españoles**: Parroquiales y civiles digitalizados
- **Buscador**: https://www.familysearch.org/search/

### Archivos Históricos Provinciales (AHP)
- Cada provincia tiene su AHP con protocolos notariales
- **Censo-guía**: http://censoarchivos.mcu.es

## Registros Civiles

| Período | Tipo de Registro |
|---------|------------------|
| Antes de 1500 | Muy pocos registros parroquiales |
| 1500-1563 | Registros parroquiales comunes |
| 1563 (Concilio de Trento) | Obligación de llevar registros parroquiales |
| 1837-1870 | Registro Civil provisional en algunas zonas |
| 1871+ | Registro Civil obligatorio nacional |

## Registros Parroquiales

| Tipo | Contenido |
|------|-----------|
| Libros de Bautismos | Nombre, fecha nacimiento/bautismo, padres, abuelos, padrinos |
| Libros de Matrimonios | Nombres de novios, padres, fecha, testigos |
| Libros de Defunciones | Nombre, edad, fecha, causa, padres/cónyuge/hijos |
| Expedientes Matrimoniales | Investigación del matrimonio (ricos en genealogía) |

## Términos Clave para Búsqueda

- `partida de bautismo` — Registro bautismal
- `partida de matrimonio` — Registro matrimonial
- `partida de defunción` — Registro de defunción
- `protocolos notariales` — Documentos notariales (testamentos, escrituras)
- `padrón` — Padrón municipal (censo)
- `quintas` — Registros militares
- `esquelas` — Necrologías en prensa
- `expediente matrimonial` — Investigación previa al matrimonio
