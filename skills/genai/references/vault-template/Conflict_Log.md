# Registro de Conflictos de Datos

## Formato de Entrada

| Fecha | Severidad | Persona | Campo | Fuente A | Fuente B | Estado | Resolución |

| | | | | | | pendiente | |

## Leyenda

### Severidad
- **CRITICAL**: Fechas de nacimiento/muerte contradictorias
- **MINOR**: Diferencias en nombres, lugares o detalles secundarios
- **COSMETIC**: Abreviaturas, ortografía, formato

### Estado
- **pendiente**: Detectado, no investigado (valor por defecto)
- **investigating**: En proceso de resolución
- **resolved**: Resuelto con fuente citada
- **escalated**: Requiere investigación adicional o decisión del usuario
